# Umm e Fatima-AI Portfolio Website

A modern, dark-themed personal portfolio website featuring a live AI chatbot powered by Google Gemini. Built with React, Tailwind CSS, and Framer Motion.

## 🌐 Live Demo
[View Live Site](https://portfolio-two-lovat-50.vercel.app)

## ✨ Features
- Live AI chatbot that answers questions about my skills, projects, and availability
- Smooth scroll animations with Framer Motion
- Custom cursor with glow effect
- Floating particle background with connecting lines
- Wave blob animations
- 3D tilt effect on skill cards
- Horizontal scrolling project cards
- Contact form with EmailJS integration
- Fully responsive design
- Deep navy and cyan color theme

## 🛠️ Tech Stack
- React.js + Vite
- Tailwind CSS
- Framer Motion
- Google Gemini AI API
- EmailJS
- Vercel (deployment)

## 🚀 Run Locally

```bash
git clone https://github.com/Umm-e-Fatima/portfolio.git
cd portfolio
npm install
npm run dev
```

## 🔑 Environment Variables
Create a `.env` file in the root folder:
VITE_GEMINI_API_KEY=your_key_here
VITE_EMAILJS_SERVICE_ID=your_key_here
VITE_EMAILJS_TEMPLATE_ID=your_key_here
VITE_EMAILJS_PUBLIC_KEY=your_key_here
